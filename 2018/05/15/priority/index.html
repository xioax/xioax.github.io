2018/05/15

CSS优先级顺序

列出一份优先级逐渐增加的选择器列表

  • 1.通用选择器。

  • 2.元素（类型）选择器。

  • 3.类选择器。

  • 4.属性选择器。

  • 5.伪类。

  • 6.Id选择器。

  • 7.内联样式。

选择器的优先权解释说明

1.内联样式表的权值最高 1000。

2.ID 选择器的权值为 100。

3.Class 类选择器的权值为 10。

4.HTML 标签（类型）选择器的权值为 1。

CSS 优先级规则：

选择器都有一个权值，权值越大越优先。

1.当权值相等时，后出现的样式表设置要优于先出现的样式表设置。

2.创作者的规则高于浏览者：即网页编写者设置的CSS 样式的优先权高于浏览器所设置的样式。

3.继承的CSS 样式不如后来指定的CSS 样式

4.在同一组属性设置中标有“!important”规则的优先级最大。
